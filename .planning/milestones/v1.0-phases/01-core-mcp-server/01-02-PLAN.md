  <action>
    1. **SkillParser.cs** -- Stateless parser for markdown with YAML frontmatter:
       - `Parse(string rawContent)` returns `(SkillFrontmatter metadata, string markdownBody, string rawContent)`.
       - Split on `---` delimiters to extract YAML block and markdown body.
       - Use `YamlDotNet.Serialization.Deserializer` to parse frontmatter into a `SkillFrontmatter` class (name, description, tags, version, etc.).
       - CRITICAL for CRUD-05: Store the ORIGINAL `rawContent` string as-is. On retrieval, return this stored raw content -- do NOT re-serialize YAML. This ensures lossless round-trip.
       - Create a `SkillFrontmatter` class/record with properties matching expected frontmatter fields: `string Name`, `string? Description`, `List<string>? Tags`, `string? Version`, plus a `Dictionary<string, object>? ExtraFields` to capture any unknown frontmatter keys without loss.
       - Handle edge cases: missing frontmatter (treat entire content as markdown body with no metadata), empty body, frontmatter-only.
       - `ToSkill(string rawContent)` convenience method that parses and returns a `Skill` model.

    2. **OpenAiEmbeddingService.cs** -- Wraps `IEmbeddingGenerator<string, Embedding<float>>`:
       - Constructor takes `IEmbeddingGenerator<string, Embedding<float>>` and `IOptions<QdrantSkillsOptions>`.
       - `GenerateEmbeddingAsync(string text, CancellationToken ct)` calls `_generator.GenerateAsync([text], ct)` and returns the float array.
       - `Dimensions` property returns configured dimensions from options.
       - Implements `IEmbeddingService` from Core.

    3. **InMemorySessionTracker.cs** -- Singleton for stdio (one process = one session):
       - Uses `ConcurrentDictionary<string, byte>` or `HashSet<string>` with lock for thread safety (per research anti-pattern warning).
       - `MarkLoaded(string skillName)` adds to set.
       - `GetLoadedSkills()` returns sorted list of loaded skill names.
       - `IsLoaded(string skillName)` checks membership.
       - `Reset()` clears all state.
       - Implements `ISessionTracker` from Core.

    4. **CollectionInitializer.cs** -- Lazy, thread-safe collection creation:
       - Constructor takes `QdrantClient` and `IOptions<QdrantSkillsOptions>`.
       - `EnsureCollectionAsync(CancellationToken ct)` -- double-checked locking with `SemaphoreSlim`:
         - If already initialized, return immediately.
         - List collections, check if target exists.
         - If not: create with `VectorParams { Size = dimensions, Distance = Cosine }`.
         - Create payload indexes: `name` (Keyword), `archived` (Bool -- verify PayloadSchemaType availability, fall back to Keyword if needed), `updated_at` (Datetime -- same fallback).
         - Set `_initialized = true`.
       - Per QDR-04: This is called before first Qdrant operation, not at startup (lazy init avoids startup failure if Qdrant is slow).

    1. **QdrantSkillRepository.cs** -- Full `ISkillRepository` implementation:
       - Constructor takes `QdrantClient`, `IOptions<QdrantSkillsOptions>`, `CollectionInitializer`, `SkillParser`, `ILogger<QdrantSkillRepository>`.
       - Static helper `GeneratePointId(string skillName)`: SHA256 hash of UTF8 bytes, take first 16 bytes, create `Guid`. Per CONTEXT.md locked decision.

       - `AddAsync(Skill skill, float[] embedding, bool overwrite, CancellationToken ct)`:
         - Call `EnsureCollectionAsync`.
         - Generate point ID from skill name.
         - If NOT overwrite: check if point exists via `RetrieveAsync` -- if exists, throw `InvalidOperationException("Skill '{name}' already exists. Use overwrite=true to replace.")`.
         - Build `PointStruct` with vectors and payload: name, description, tags (as array), raw_content (FULL original markdown), archived=false, updated_at=now (ISO 8601).
         - `UpsertAsync`.

       - `UpdateAsync(Skill skill, float[] embedding, CancellationToken ct)`:
         - Call `EnsureCollectionAsync`.
         - Generate point ID, verify skill exists (throw if not found).
         - Upsert with new content and updated_at.

       - `DeleteAsync(string skillName, CancellationToken ct)`:
         - `DeleteAsync` by point ID (from skill name hash).

       - `ArchiveAsync(string skillName, CancellationToken ct)`:
         - Set `archived: true` in payload via `SetPayloadAsync`.

       - `GetByNameAsync(string skillName, CancellationToken ct)`:
         - `RetrieveAsync` by point ID.
         - Reconstruct `Skill` from payload (return `raw_content` as-is for lossless round-trip).

       - `SearchAsync(float[] queryEmbedding, int maxResults, float? scoreThreshold, CancellationToken ct)`:
         - Call `EnsureCollectionAsync`.
         - Build `SearchParams` with the query vector.
         - Filter: `archived != true` (exclude archived skills).
         - Limit: maxResults. ScoreThreshold: scoreThreshold if provided.
         - Map results to `SearchResult` list, ordered by score desc (Qdrant default), with recency tiebreaker via `updated_at` (use `OrderByDescending(score).ThenByDescending(updated_at)` post-query if Qdrant doesn't support composite ordering).

       - `ListAsync(CancellationToken ct)`:
         - `ScrollAsync` with filter `archived != true`.
         - Return `SkillMetadata` list (name, description, tags, score=0, updated_at).

       - `EnsureCollectionAsync(CancellationToken ct)`:
         - Delegates to `CollectionInitializer.EnsureCollectionAsync`.

    2. **ServiceRegistration.cs** -- Extension method `AddQdrantSkillsInfrastructure(this IServiceCollection services, IConfiguration config)`:
       - Bind `QdrantSkillsOptions` from `config.GetSection("QdrantSkills")`.
       - Register `QdrantClient` as singleton (created from options: host, port, API key).
       - Register `CollectionInitializer` as singleton.
       - Register `SkillParser` as singleton (stateless).
       - Register `QdrantSkillRepository` as `ISkillRepository` (singleton).
       - Register `OpenAiEmbeddingService` as `IEmbeddingService` (singleton).
       - Register `IEmbeddingGenerator<string, Embedding<float>>` using `new OpenAIClient(apiKey).AsEmbeddingGenerator(modelId: options.EmbeddingModel)`.
       - Register `InMemorySessionTracker` as `ISessionTracker` (singleton -- one process = one session for stdio).
  </action>
