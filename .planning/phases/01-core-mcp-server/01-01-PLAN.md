    3. Create `src/QdrantSkillsMCP.Core/QdrantSkillsMCP.Core.csproj` -- ZERO NuGet packages, only framework reference.

    4. Create models in `src/QdrantSkillsMCP.Core/Models/`:
       - `Skill.cs` -- record with: `string Name`, `string? Description`, `string[]? Tags`, `string RawContent` (original markdown+frontmatter), `string MarkdownBody` (body only), `DateTimeOffset UpdatedAt`, `bool Archived`. Name is the primary key.
       - `SkillMetadata.cs` -- record with: `string Name`, `string? Description`, `string[]? Tags`, `float Score`, `DateTimeOffset UpdatedAt`. Used for `includeContent: false` responses.
       - `SearchResult.cs` -- record with: `Skill Skill`, `float Score`. Wrapper for search results with similarity score.

    5. Create interfaces in `src/QdrantSkillsMCP.Core/Interfaces/`:
       - `ISkillRepository.cs`:
         - `Task AddAsync(Skill skill, float[] embedding, bool overwrite, CancellationToken ct)`
         - `Task UpdateAsync(Skill skill, float[] embedding, CancellationToken ct)`
         - `Task DeleteAsync(string skillName, CancellationToken ct)`
         - `Task ArchiveAsync(string skillName, CancellationToken ct)`
         - `Task<Skill?> GetByNameAsync(string skillName, CancellationToken ct)`
         - `Task<IReadOnlyList<SearchResult>> SearchAsync(float[] queryEmbedding, int maxResults, float? scoreThreshold, CancellationToken ct)`
         - `Task<IReadOnlyList<SkillMetadata>> ListAsync(CancellationToken ct)`
         - `Task EnsureCollectionAsync(CancellationToken ct)`
       - `IEmbeddingService.cs`:
         - `Task<float[]> GenerateEmbeddingAsync(string text, CancellationToken ct)`
         - `int Dimensions { get; }` (for collection creation)
       - `ISessionTracker.cs`:
         - `void MarkLoaded(string skillName)` -- marks a skill as loaded in current session
         - `IReadOnlyList<string> GetLoadedSkills()` -- returns all loaded skill names
         - `bool IsLoaded(string skillName)` -- checks if skill was already loaded
         - `void Reset()` -- clears session state

    6. Create `src/QdrantSkillsMCP.Core/Validation/SkillNameValidator.cs`:
       - Static class with `Validate(string name)` returning `(bool IsValid, string? Error)`.
       - Rules: lowercase letters, numbers, hyphens only; max 64 chars; cannot start/end with hyphen; cannot be empty.
       - Use regex: `^[a-z0-9]([a-z0-9-]{0,62}[a-z0-9])?$` (handles 1-char and 2+ char names).

    7. Create the solution file `QdrantSkillsMCP.sln` using `dotnet new sln` and add the Core project.
